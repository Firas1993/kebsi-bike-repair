# Kebsi Bike Repair - Professional Bicycle Repair Services

A professional website for Kebsi Bike Repair, a bicycle repair service located in Borj Cedria, Tunis, managed by Idriss Kebsi.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Service Showcase**: Displays all available bike repair services
- **Online Booking System**: Customers can book appointments with preferred time slots
- **Mobile Service**: Option for customers to request doorstep service
- **Image Upload**: Customers can upload photos of bike issues
- **Modern UI**: Clean, professional design with smooth animations
- **Contact Information**: Easy access to location and contact details

## Services Offered

- General bike repairs and tune-ups
- Brake services and adjustments
- Wheel and tire repairs
- Chain and gear maintenance
- All types of bikes (Mountain, Road, Electric, BMX, etc.)
- Mobile repair service (we come to you!)

## Setup Instructions

1. **Download all files** to a folder on your computer:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`

2. **Add the logo image**:
   - Save the Kebsi Bike Repair logo as `logo.png` in the same folder

3. **Open the website**:
   - Double-click on `index.html` to open it in your web browser

## Customization

### Adding Your Logo
Replace `logo.png` with your actual logo file. The logo should be:
- Format: PNG (with transparent background recommended)
- Size: 200x200px or larger (it will be automatically resized)

### Updating Contact Information
Edit the contact section in `index.html` to add:
- Phone number
- Email address
- Social media links
- Exact address details

### Backend Integration
To make the booking form functional, you'll need to:
1. Set up a backend server (Node.js, PHP, Python, etc.)
2. Create an API endpoint to receive form submissions
3. Uncomment the fetch code in `script.js` and update the endpoint URL
4. Set up email notifications or database storage
